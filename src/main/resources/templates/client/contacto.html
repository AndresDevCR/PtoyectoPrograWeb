Acerca Nosotros

Historia

II Cuatrimestre del 2022

Esta página es el proyecto de Desarrollo de Aplicaciones Web y Patrones que se empezó a trabajar desde el comienzo de este cuatrismestre con el profesor Don Wilberth Molina Perez.

Esta página trata de no solo ser un proyecto que realizar. Solo Llantas CR tiene como objetivo ser una de las mejores, o sino la mejor página Web relacionada con todo lo que tenga que ver con vehículos en Costa Rica.

Proposito

Se desea ingresar al mundo real, ser competencia hacía las demas páginas de Costa Rica. El proposito es ayudar a los usuarios con necesidades relacionadas con sus vehículos o motocicletas, si el cliente desea vender su vehiculo, alquilar un vehiculo por determinado tiempo, comprar un vehiculo usado entre otras cosas, Solo Llantas CR le proporciona esas posibilidades a los usuarios.

Herramientas Utilizadas

A continuación se mostrar algunas de las herramientas que se utilizaron para el desarrollo de Solo Llantas CR:

HyperText Markup Language (HTML)
Es el código que se utiliza para estructurar y desplegar una página web y sus contenidos.
Cascading Style Sheets (CSS)
Es un lenguaje que maneja el diseño y presentación de las páginas web.
Java
Es una plataforma informática de lenguaje de programación.
My Structured Query Language(MySQL)
Este permite almacenar y acceder a los datos a través de múltiples motores de almacenamiento.
